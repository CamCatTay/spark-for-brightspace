Frequently Asked Questions

Nothing is showing up / The calendar is not loading

Refresh the page you are on or reinstall the extension. If the problem persist please submit a bug report so this issue can be looked into.

Why are some of my assignments not showing up?

Spark is a third-party browser extension and operates within the permissions granted by your institution's Brightspace instance. Certain course materials, such as items restricted by availability dates, access conditions, or course-level permissions, may not be retrievable without direct authorization from Brightspace. As a result, some assignments may not appear in the panel.

The link to my assignment isn't working. Why?

Some assignment links are generated dynamically by Brightspace and require an authenticated session with specific course-level permissions to resolve correctly. If a link opens an error page or redirects unexpectedly, this is a limitation of how Brightspace exposes that content to third-party integrations and is not something Spark can work around directly. Navigating to the assignment from within Brightspace directly should always work as expected.

How do I report a bug or request a new feature?

Bug reports and feature requests are tracked on GitHub. You can open an issue using the link below please include as much detail as possible, such as what you expected to happen, what actually happened, and your browser version.
